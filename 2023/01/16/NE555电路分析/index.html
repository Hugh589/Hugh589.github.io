Multisim分析NE555

一、电路分析

NE555共有8个输入端，其中，1端接地，2端低触发端，又称触发端；3端输出端；4端清零端，低电平有效；5端为控制电压端，可改变比较器的基准电压，该端不用时，应接入一个0.01uF的电容；6端高触发端，又称阈值端；7端为放电端；8端为Vcc端。

##二、Multisim仿真

###1、电路模块仿真

（1）单稳触发器

  • 电路工作原理

    单稳触发器的电路如下图所示：

    ​ 出现触发信号时，2端口为L，C2输出为L，5端口为H，C1输出为H；RS触发器输出为L，Vo输出为H。同时，三极管截止，电源Vcc通过RC对6端口充电。若在此期间，触发信号消失，即2端口变为H，此时R=H，S=H，触发器保持，继续充电，当充电超过$\frac{2}{3}Vcc$时，R=L，S=H，Vo为L，三极管导通，6端口开始通过三极管放电，使电路进入稳态。根据电路的三要素法可求出单稳态电路的周期：

    $$
    f(t) = f(\infty) + [f(0) + f(\infty)]*e^{t/t_w}
    $$
    在本电路中，$t_w = RC,f(\infty) = Vcc,f(0) = 0,f(t) = \frac{2}{3}Vcc$ ,故可求出$t = RC\ln3 = 1.1RC$。

（2）多谐触发器

  • 电路工作原理

    ​ 多谐振荡电路的内部结构如下图所示：

    ​ 在刚刚接通电源的时候，2端口的电压为0，此时，R=H,S = L,Vo=H；然后电源通过R1和R2对电容充电至$\frac{2}{3}Vcc$,此时，R=L,S=H，Vo=L,三极管导通，电容通过R2放电至$\frac{1}{3}Vcc$,然后继续充电，重复上述过程。

    ​ 因此，可以通过三要素法对充放电的时间进行计算。充电时,$t_w = (R_1+R_2)C,f(\infty) = Vcc,f(0) = \frac{1}{3}Vcc,f(t) = \frac{2}{3}Vcc$,故可求出$t_1 = (R_1+R_2)C\ln2 = 0.7(R_1+R_2)C$。放电时，$t_w = R_2C,f(\infty) = 0,f(0) = \frac{2}{3}Vcc,f(t) = \frac{1}{3}Vcc$,故可求出$t_1 = R_2C\ln2 = 0.7R_2C$。

  • 电路仿真

    从示波器中可以看出，输出的高电平的持续时间为610.795us。
